## 🎯 Project Overview

This is a lightweight, responsive **Simple Shopping List Application** built using pure Web technologies. It serves as an excellent reference point for students and boot camp learners who want to see how to manipulate HTML elements dynamically using JavaScript.

### ✨ Key Features
*   **Dynamic Item Addition:** Add items to your shopping list smoothly.
*   **Easy Item Removal:** Remove items instantly with a click.
*   **Clean & Responsive UI:** Optimized layout that looks great on mobile and desktop.
*   **Beginner-Friendly Codebase:** Well-structured, commented code designed for easy learning.

---

## 🚀 Tech Stack & Core Concepts

*   **HTML5:** Semantic structure for web accessibility.
*   **CSS3:** Clean layout, modern spacing, and responsive design.
*   **Vanilla JavaScript (ES6+):** 
    *   **DOM Manipulation:** Dynamically creating, appending, and deleting elements.
    *   **Event Handling:** Listening for user clicks and form submissions.
